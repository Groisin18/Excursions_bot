# Структура роутеров

## Порядок обработки:
1. admin. раздел с роутерами админ-панели (с middleware)
1.1. `admin_router.py` - основные админские команды
1.2. `bookings.py` - запись клиентов на экскурсию
1.3. `captains.py` - управление капитанами
1.4. `clients.py` - управление клиентами
1.5. `excursions.py` - управление видами экскурсий
1.6. `finances.py` - управление финансами
1.7. `notification.py` - отправка уведомлений
1.8. `promocodes.py` - управление промокодами
1.9. `schedule.py` - управление расписанием экскурсий
1.10. `settings.py` - общий настройки
1.11. `statistic.py` - статистика
2. `payment_router.py` - оплата пользователем экскурсии
3. `registration_router.py` - регистрация пользователей
4. `redaction_router.py` - редактирование данных
5. `user_router.py` - основные пользовательские команды
6. `fallback_router.py` - обработка неизвестных команд

## Правила именования:
- Файлы: `{сущность}_router.py`
- Роутер: `{сущность}_router`
- Имя роутера: `router = Router(name="{сущность}")`
