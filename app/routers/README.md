# Структура роутеров

## Порядок обработки:
1. admin. раздел с роутерами админ-панели (с middleware)
1.1. `admin_main_router.py` - основные админские команды
1.2. `bookings.py` - запись клиентов на экскурсию
1.3. `captains.py` - управление капитанами
1.4. `clients.py` - управление клиентами
1.5. `excursions.py` - управление видами экскурсий
1.6. `finances.py` - управление финансами
1.7. `notification.py` - отправка уведомлений
1.8. `promocodes.py` - управление промокодами
1.9. `schedule.py` - управление расписанием экскурсий
1.10. `new_slot.py` - добавление нового слота
1.11. `slots.py` - управление имеющимися слотами (редактирование, перенос, назначение капитанов, удаление)
1.12. `settings.py` - общие настройки
1.13. `statistic.py` - статистика
2. user. раздел с роутерами юзер-панели
2.1 `test_payment.py` - оплата пользователем экскурсии
2.2 `registration.py` - регистрация пользователей
2.3 `redaction_userdata.py` - редактирование данных
2.4 `user_main.py` - основные пользовательские команды
3. `fallback.py` - обработка неизвестных команд

## Правила именования:
- Файлы: `{сущность}.py`
- Роутер: `{сущность}_router`
- Имя роутера: `router = Router(name="{сущность}")`
